## Description

[NestRestApi] Une application offrant des API REST pour un usage personnel

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
